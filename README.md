# INTRODUCCIÓN
# Existen 2 tipos fundamentales de producto: 
# •	Destinado a satisfacer necesidades personales directas: Alimentos, joyas, etc. Estos son bienes y servicios de consumo.
# •	Destinado a satisfacer necesidades de producción de empresa: Maquinaria, equipos, etc. Estos son bienes y servicios de capital.

# MEDICIÓN DE LA ACTIVIDAD ECONÓMICA GLOBAL: 
# Bienes de capital tienen la capacidad de ser consumidos, por ello se necesita un sacrificio de una parte del consumo. 
# Proceso de crear capital es inversión (no confundir con acciones o bonos que no son inversión).
# En economía el dinero no es bien de capital, porque no es factor de producción, por ende el incremento de este no aumentará el PBI o riqueza del país. 
# El dinero es un medio que hace posibles intercambios comerciales, pero por si solo no puede aumentar la producción.
# Ejem: Para sumarse la producción de autos, viviendas, alimentos, etc. se hará mediante valores monetarias para contabilizarlos.
# Nota: La producción de BYS son medibles en términos monetarios
# Los BYS se venden mediante precios o valores monetarios.
